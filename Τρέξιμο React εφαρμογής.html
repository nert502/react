Για να τρέξεις μια εφαρμογή React, πρέπει να έχεις πρώτα δημιουργήσει ένα project (π.χ. με create-react-app) και να έχεις εγκατεστημένο το Node.js στο σύστημά σου.

✅ Βήματα για να τρέξεις μια εφαρμογή React:

1. Δημιουργία νέου project (αν δεν έχεις ήδη):

bash
npx create-react-app my-app
cd my-app

2. Εκκίνηση του React app:

bash
npm start

📌 Η παραπάνω εντολή ανοίγει αυτόματα τον browser και φορτώνει την εφαρμογή σου στο http://localhost:3000.

💡 Extra Tips:

  • Αν χρησιμοποιείς yarn αντί για npm:

    bash
    yarn start

  • Αν θες να δεις αν έχεις Node εγκατεστημένο:

    bash
    node -v

  • Αν πάρεις κάποιο σφάλμα, στείλε μου το μήνυμα για να σε βοηθήσω.
